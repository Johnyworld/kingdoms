# Feature: Selection & Movement (선택과 이동)

> 스크립트: `scenes/game/game.gd`, `scenes/game/range_overlay.gd`

주인공을 선택하면 이동/공격 범위가 표시되고, 범위 안의 칸을 클릭해 이동한다.

## 좌클릭 처리 (`_handle_click`)

우선순위 순서:

1. **캠프 클릭이 최우선** — 선택 중이면 해제하고, [캠프 메뉴](camp-menu.md)를 연다.
2. **선택 안 됨** — 주인공이 선 칸을 클릭하면 선택(`_select`).
3. **선택됨**:
   - 이동 범위(거리 1 ~ 이동력) 칸이면 그 칸으로 이동 후 안개 갱신.
   - 그 외 칸이면 선택 해제.

## 범위 계산 (`_update_ranges`)

- 주인공 위치에서 **BFS**로 도달 셀과 거리를 계산.
- **이동 범위**: 거리 1 ~ `movement` → 파랑 헥스.
- **공격 범위**: 거리 `movement + 1` (마지막 링) → 빨강 헥스. 이 링에서는 더 확장하지 않는다.
- 주인공이 선 칸(거리 0)은 제외.

## 범위 오버레이 (`range_overlay.gd`)

> `extends Node2D`

- `setup(terrain)` — 지형 참조 설정.
- `show_ranges(move_cells, attack_cells)` — 표시할 셀을 갱신하고 다시 그린다.
- 색상: 이동 `Color(0.2, 0.5, 1.0, 0.35)` 파랑 / 공격 `Color(1.0, 0.25, 0.25, 0.35)` 빨강.
- 각 셀에 타일 크기에 맞춘 헥스 폴리곤을 채워 실제 타일과 겹치게 그린다.

## 선택 상태

- `_select()` — 선택 플래그 on, 주인공 강조 링 표시, 범위 계산.
- `_deselect()` — 플래그 off, 강조 링 제거, 범위 표시 지움.
