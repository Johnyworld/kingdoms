# Feature: Combat Resolution (전투 판정 로직)

> 스크립트: `scenes/combat/combat_resolver.gd` (`class_name CombatResolver extends RefCounted`)

두 [부대](../entities/Party.md)의 [멤버](../entities/Human.md)가 벌이는 교전을 **능력치로 판정하는 순수 로직**이다. 씬·시각 요소 없이 데이터만 다뤄 시드 RNG로 결정적으로 테스트한다(HexGrid·ClickRouter와 같은 헬퍼 패턴).

기획 원본 `docs/table/시스템/전투.md`(랑그릿사식)의 규칙 중 **현재 구현된 능력치만으로 가능한 부분**을 옮긴 첫 슬라이스다. 전투씬·이동·타겟팅·사상자 반영은 다음 슬라이스(미구현).

## 계산 스탯

현재 [능력치](../data/stats.md)만 사용한다. 장비·지형이 없어 관련 항은 0/생략이다.

| 값 | 식 | 비고 |
| --- | --- | --- |
| 공격력 AT | `floor(힘 / 5)` | 무기 공격력 `미구현`(0) |
| 방어력 DF | `0` | 방어구·방패 `미구현` |
| 회피율(%) | `민첩 × 0.5` | 지형·장비무게 보정 `미구현` |
| 명중(%) | `90 − 회피율` | 상한 clamp 없음 — 0 이하면 무조건 빗나감 |
| 치명타(%) | `행운 × 0.5` | |

## 1회 공방 판정 (`resolve_hit`)

한 번의 타격을 다음 순서로 처리한다. `rng: RandomNumberGenerator`로 확률을 굴린다.

1. **명중 판정** — `rng.randf()×100 < 명중(%)`이면 명중. 빗나가면 피해 0.
2. **치명타 판정** — 명중 시 `rng.randf()×100 < 치명타(%)`이면 치명타(배율 1.5), 아니면 1.0.
3. **피해** — `피해 = floor(max(1, AT − DF) × 치명배율)`.
4. **생명점 차감** — `대상 생명점 − 피해`. 0 이하면 전투불능(사망).

- 반환: `{hit, crit, damage, hp(차감 후), dead}`.
- 방패 막기·상성배율·상태이상 부여는 `미구현`(장비·상태이상 시스템 없음).

## 교전 (`resolve_engagement`)

근접 교전만 다룬다(원거리·리치 선제권 `미구현`).

- **개시자(a)가 선공**한다. `a→b`, `b→a`를 한 라운드로 **최대 3라운드**(각자 최대 3타) 반복한다.
- 각 타격은 `resolve_hit`를 따른다.
- 도중에 한쪽 생명점이 0 이하가 되면 **즉시 종료**한다(그 라운드의 남은 반격 없음 → 선공 이점).
- 반환: `{a_hp, b_hp, a_dead, b_dead}`.

## 미구현 (다음 슬라이스/시스템)

- **전투씬·유닛 이동·타겟팅·목표 재탐색·사상자 반영·월드맵 복귀** — [Battle](battle.md)에서 이 판정 로직을 사용해 구현한다.
- **무기·방어구·방패·공격거리/리치·원거리 공격** — 아이템 시스템 없음.
- **마법·상성배율·상태이상·치명타 연동 상태이상** — 관련 시스템 없음.
- **지형 전투 보정(방어·회피·시야)·지휘관 지휘범위 보너스·스태미나·사기 보정** — `미구현`.

## 테스트 시나리오

`test/unit/test_combat_resolver.gd`. Human은 능력치를 직접 설정해 만든다.

### 계산 스탯 (결정적, RNG 없음)
- [정상] `attack_power` = `floor(힘/5)` — 힘 78 → 15, 힘 4 → 0
- [정상] `evasion` = `민첩 × 0.5`
- [정상] `hit_chance` = `90 − 대상 회피율`
- [정상] `crit_chance` = `행운 × 0.5`
- [정상] `hit_damage(공격자, crit)` = `floor(max(1, AT) × (1.5 if crit else 1.0))` — AT 0이어도 최소 1

### 1회 공방 (`resolve_hit`)
- [경계] 회피율이 매우 높아(민첩 200 → 회피 100 → 명중 −10) 항상 빗나감 → `hit=false`, 피해 0, hp 불변
- [정상] 같은 시드 → 같은 결과(결정적)
- [정상] 명중 시 `hp = 이전 hp − damage`, `dead = hp ≤ 0`

### 교전 (`resolve_engagement`)
- [경계] 양쪽 모두 회피 100(항상 빗나감) → 3라운드 모두 빗나가고 양측 hp 불변·생존
- [정상] 공격자가 압도적(피해 ≥ 대상 생명점)이고 명중하는 시드 → 대상 전투불능, **개시자는 무사**(선공 이점, 반격 없음)
- [불변] 결과 hp는 초기 hp를 넘지 않는다(피해는 감소만)
- [정상] 같은 시드 → 같은 결과(결정적)
