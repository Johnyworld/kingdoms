# 전투

무기·방어구·방패·상태이상·마법·지형·유닛 스탯이 만나는 계산 규칙을 정의한다.
개별 수치는 각 문서에, **결합 방식**은 이 문서에 둔다.

> **벤치마크: 랑그릿사 1편.** 월드맵에서 [부대](docs/table/세력/부대.md)가 부대를 공격하면 전투씬으로 전환되고,
> 양측 부대원이 각자 개별 교전을 벌인다.

## 전투의 흐름

### 1. 개시 (월드맵)

- 월드맵에서 부대 A가 사거리(= 유닛 [무기](docs/table/아이템/무기.md) `공격거리`) 안의 부대 B를 공격하면 **전투씬**으로 전환된다.
- 전투는 **부대 단위**로 벌어진다. 한 부대는 최대 **10명**(지휘관 [지휘력](docs/table/세력/유닛.md)에 따라 가변).

### 2. 공격 유형

공격을 건 **거리**에 따라 두 방식으로 진행된다.

| 유형    | 조건            | 진행                                          |
| ----- | ------------- | ------------------------------------------- |
| 근접 교전 | 인접 헥스에서 공격    | 양측 부대원이 돌진해 개별 교전                           |
| 원거리 공격 | 사거리를 둔 채 공격 | 공격 측의 **원거리 유닛만** 발사, 나머지 근접 유닛은 대기         |

### 3. 근접 교전

- 양측 부대원이 필드에 배치되고, 각 유닛이 **개별 AI**로 움직인다.
- 각 유닛은 가장 가까운(또는 유효한) 적에게 **돌진**해 교전한다.
- 두 유닛이 접촉하면 교전이 성립하며, **리치가 긴 무기**(근접거리 ↑)를 든 쪽이 **선제 공격**한다.
- 한 교전에서 두 유닛은 **각각 3회씩 공방을 주고받는다** (공방 횟수 **고정** N = 3).
  - 선제 측 1타 → 후공 측 1타 → … 를 3회 반복.
  - 도중에 한쪽 생명점이 0이 되면 교전 즉시 종료.
- 매 타격은 아래 **[1회 공방 판정](#1회-공방-판정)**을 따른다.

### 4. 원거리 공격

- **공격 측**: 부대 내 **원거리 무기(활·투창·마법)** 보유 유닛만 공격한다. 근접 유닛은 대기.
- **방어 측(반격)**: **사거리가 닿는 원거리 유닛만** 반격한다. 닿는 유닛이 없으면 반격 없이 일방적으로 피해를 입는다.
- 원거리 공방도 유닛당 **3회 고정** 규칙을 따른다.

### 5. 목표 재탐색 · 종료

- 교전으로 **사망자가 나오면**, 생존한 유닛은 즉시 **다음 목표**를 찾아 이동·교전한다.
- 전투는 **한쪽 전멸까지가 아니라**, 정해진 교전 처리가 끝나면 종료된다.
- 회피·공격력 차이에 따라 **양측 무사**할 수도, **한쪽 전멸**할 수도 있다.
- 종료 시 양측 사망자/생존자를 집계하고, 생존 부대원은 월드맵으로 복귀한다.

## 지휘관 보정 (지휘 범위)

- 지휘관의 [지휘력](docs/table/세력/유닛.md)에 따라 **지휘 범위**가 정해지며, 그 안의 아군 부대원은 능력 보너스(공격력·방어력·명중 등)를 받는다.
- 지휘 범위를 벗어나거나 **지휘관이 사망**하면 해당 보너스를 잃는다. (랑그릿사식 지휘 시스템)

## 계산 스탯 (유닛 "계산됨" 항목)

[유닛](docs/table/세력/유닛.md)의 특성 + 장비 + [지형](docs/table/시스템/타일.md) 보정으로 산출한다. (전투씬 UI의 **AT / DF / 지형** 표기에 대응)

```
공격력(AT) = 무기 공격력 + floor(힘 / 5)          # 마법 무기는 아래 "마법" 참조
방어력(DF) = Σ(착용 방어구 방어력) + 방패 방어력
회피율(%)  = 민첩 × 0.5 + 지형 회피 보정 − 총 장비무게 × 0.3   # UI의 "지형" %
공격범위   = 무기 공격거리
근접거리   = 무기 근접거리                          # 교전 선제권 결정
이동력    = 탈것 이동력 + 지형 이동력 보정            # 도로 +2, 언덕 −1
시야     = 기본 시야 + 지형 시야 보정 (+ 건물 시야)
```

- 방어/회피에는 지형 보정이 더해진다. (숲 방어 +2·시야 −2 등, [타일](docs/table/시스템/타일.md))

## 피해 계산

### 물리 (참격 · 자돌 · 타격 · 원거리)

```
기본피해 = max(1, 공격력 − 방어력)
최종피해 = 기본피해 × 상성배율 × 치명타배율
```

- **상성배율**: 공격의 데미지 타입 × 방어구 분류 → [방어구](docs/table/아이템/방어구.md) 상성표.
- **치명타배율**: 치명타 시 1.5, 평타 1.0.

### 마법

```
마법 위력 = 무기(촉매) 공격력 + 시전자의 해당 속성 마법 기능치
최종피해  = 마법 위력 × 상성배율
```

- 마법은 물리 방어력을 무시하고, 방어구 분류의 **마법 상성배율**만 적용한다. (판금 1.3 취약 / 천 0.6 경감)
- 속성별 스킬·상태이상은 [마법](docs/table/마법/불마법.md) 문서에서 정의.

## 1회 공방 판정

교전 중 한 번의 타격은 다음 순서로 처리한다.

1. **명중 판정** — `명중 = 기본명중(90) − 대상 회피율`. 실명 시 명중 −50%.
2. **방패 막기** — 명중 시, 대상의 [방패](docs/table/아이템/방패.md) `막기 확률`로 판정. 성공하면 **피해 완전 무효**.
3. **치명타 판정** — `치명타 확률 = 행운 × 0.5(%)`. 성공 시 배율 적용 + 상태이상 부여.
4. **피해 적용** — 위 피해 계산식으로 생명점 차감. 0이면 사망.
5. **상태이상 부여** — 치명타/속성 효과에 따라 [상태이상](docs/table/시스템/상태이상.md) 부여.

### 치명타 → 상태이상 연동

| 데미지 타입 | 치명타 시 부여 |
| ------ | -------- |
| 참격     | 출혈       |
| 타격     | 기절       |
| 자돌     | (방어력 관통) |
| 원거리    | -        |
| 마법     | 속성별 효과   |

## 스태미나

- 월드맵 이동 시 [타일](docs/table/시스템/타일.md)의 `스태미나 소모(육지)`만큼 감소한다. (하늘 2, 배 3 고정)
- 스태미나가 낮으면 전투 성능이 떨어지고, 0이면 행동 불가. 휴식·[원기 물약](docs/table/아이템/소모품.md)으로 회복.

## 사기

- **낮은 사기**: 공격력·명중 페널티, 임계 이하 시 확률적 도주.
- **높은 사기**: 소폭 보너스.
- 전투 승리 시 사기 +, 패배·아군 사망 시 −. 다양한 음식·[용기의 술](docs/table/아이템/소모품.md)로 회복. ([유닛](docs/table/세력/유닛.md) 사기 규칙)
- 공포 상태이상은 사기를 크게 깎고 도주 확률을 높인다.
