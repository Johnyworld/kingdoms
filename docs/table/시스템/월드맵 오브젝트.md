# 월드맵 오브젝트

[타일](docs/table/시스템/타일.md) 위에 놓이는 상호작용 요소. 지형(고정 배경)이나 [건물](docs/table/세력/건물.md)(플레이어 건설물)과 구분된다.

### 속성

| 속성명   | 타입                                    | 설명                    |
| ----- | ------------------------------------- | --------------------- |
| 이름    | string                                |                       |
| 분류    | 자원 노드 \| 채집물 \| 유적 \| 중립 \| 위협        |                       |
| 등장 지형 | [타일](docs/table/시스템/타일.md)            | 주로 나타나는 지형            |
| 상호작용  | string                                | 부대·건물이 이 오브젝트에 하는 행동  |
| 산출/효과 | string                                | 획득 자원·아이템·이벤트         |
| 재생    | boolean                               | 소진 후 시간이 지나 다시 생기는지   |

### 일람

| 분류    | 이름       | 등장 지형   | 상호작용         | 산출/효과                | 재생  |
| ----- | -------- | ------- | ------------ | -------------------- | --- |
| 자원 노드 | 철광맥      | 언덕·산    | 철광 건설        | 철 채굴 지점              | X   |
| 자원 노드 | 금광맥      | 언덕·산    | 금광 건설        | 금 채굴 지점              | X   |
| 자원 노드 | 은광맥      | 언덕·산    | 은광 건설        | 은 채굴 지점              | X   |
| 자원 노드 | 원시림      | 숲       | 벌목소 건설       | 나무 산출 증가             | O   |
| 자원 노드 | 어장       | 물·해안    | 낚시터 건설       | 생선 산출 증가             | O   |
| 채집물   | 보석 원석    | 산·언덕    | 부대 채집        | 보석(루비·사파이어·에메랄드 등)   | X   |
| 채집물   | 약초 군락    | 숲·습지    | 부대 채집        | 물약 재료·소량 식량          | O   |
| 채집물   | 야생 모피    | 설원·숲    | 부대 채집        | 모피                   | O   |
| 채집물   | 소금밭      | 사막·해안   | 부대 채집        | 소금                   | O   |
| 채집물   | 향신료 자생지  | 사막·숲    | 부대 채집        | 향신료                  | O   |
| 유적    | 폐허       | 임의      | 부대 탐사        | 자원·아이템 (탐사 이벤트)      | X   |
| 유적    | 고대 유적    | 임의      | 부대 탐사        | 희귀 아이템·상아·다이아몬드      | X   |
| 유적    | 보물 상자    | 임의      | 부대 획득        | 금·아이템               | X   |
| 중립    | 상인 카라반   | 도로·평원   | 접촉 시 거래      | 이동식 상점 (사치품 매매)      | -   |
| 중립    | 중립 마을    | 평원      | 교역·회유       | 자원 거래, 회유 시 인구·유닛 합류 | -   |
| 위협    | 야수 소굴    | 숲·산     | 전투           | 처치 시 자원·모피·경험 보상     | X   |
| 위협    | 도적 캠프    | 임의      | 전투           | 처치 시 약탈 자원·아이템       | X   |

### 규칙

- **자원 노드**는 그 위/인접에 대응 [건물](docs/table/세력/건물.md)을 지어야 산출을 얻는다. (광맥→광산, 원시림→벌목소, 어장→낚시터)
- **채집물**은 부대가 해당 헥스에서 채집하며, 다수는 [사치품](docs/table/아이템/사치품.md)·[자원](docs/table/아이템/자원.md)으로 귀속된다.
- **유적**의 탐사 결과는 [이벤트](docs/table/시스템/이벤트.md)의 탐사 결과표를 따른다.
- **위협**은 중립 적대 [부대](docs/table/세력/부대.md)로 취급되어 [전투](docs/table/시스템/전투.md)로 해결한다.
