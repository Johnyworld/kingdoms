# 상태이상

유닛에게 일정 턴 동안 부여되는 효과. 이로운 효과(버프)와 해로운 효과(디버프)를 포함한다.
생명점·스태미나·사기 등 [유닛](docs/table/세력/유닛.md) 스탯에 작용한다.

### 속성

| 속성명   | 타입          | 설명                          |
| ----- | ----------- | --------------------------- |
| 이름    | string      |                             |
| 분류    | 이로움 \| 해로움  |                             |
| 효과    | string      | 스탯 변화 · 행동 제약               |
| 지속    | number      | 지속 턴                        |
| 중첩    | boolean     | 재부여 시 효과 누적 여부              |
| 해제 방법 | string      | 자연 소멸 외 강제 해제 수단            |
| 주 원인  | string      | 이 효과를 유발하는 무기 · 마법 · 소모품    |

### 일람 — 해로움

| 이름  | 효과             | 지속  | 중첩  | 해제 방법        | 주 원인               |

| 출혈  | 턴당 생명점 −4      | 2   | O   | 붕대 · 치료 물약   | 참격 무기 치명타          |

### 규칙

- **지속**은 부여된 턴 다음 턴부터 감소하며, 0이 되면 해제된다.
- **중첩 O**(화상·중독·출혈·공포)는 재부여 시 지속 턴이 갱신되고 도트 피해가 누적된다.
- **정화 수단**: [정화 물약·해독제·만병통치약](docs/table/아이템/소모품.md), [신전](docs/table/세력/건물.md)(영지 단위 정화).
- **치명타 유발**: 참격 → 출혈, 타격 → 기절. [무기](docs/table/아이템/무기.md)의 데미지 타입과 연동.
- 각 마법 계열이 유발하는 상태이상은 개별 [마법](docs/table/마법/불마법.md) 문서에서 정의한다.
