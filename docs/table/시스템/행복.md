# 행복

**영지(인구) 단위의 만족도.** 유닛·부대 단위의 [사기](docs/table/세력/유닛.md)와는 층위가 다르며, 서로 연동된다.

| 구분 | 행복 | 사기 |
| --- | --- | --- |
| 대상 | 영지 · 인구 | 유닛 · 부대 |
| 성격 | 정착민의 삶의 질 | 전투 의욕 |
| 범위 | 0 ~ 100 | 0 ~ 100 |

- 술집·다양한 음식 등 일부 요소는 행복과 사기를 **동시에** 올린다.

### 속성 (영지에 귀속)

| 속성명    | 타입     | 설명                     |
| ------ | ------ | ---------------------- |
| 행복     | number | 0 ~ 100, 기본값 50        |
| 생산 배율  | number | 행복으로부터 계산됨 (0.8 ~ 1.2) |

## 행복의 소비처 (행복이 영향을 주는 것)

| 대상          | 효과                                       |
| ----------- | ---------------------------------------- |
| 인구 증감       | 높으면 여유인구 자연 증가(이주·회복), 낮으면 이탈·감소         |
| 생산 효율       | 건물 `턴당 생산`에 배율 적용                        |
| 세수 (금·은)    | 영지 재화 수입에 비례                              |
| 징집 병력 초기 사기 | 여유인구를 병사로 전환할 때 시작 [사기](docs/table/세력/유닛.md)값 |
| 반란 · 이탈     | 극도로 낮으면 총독 이탈 · 건물 파업 · 영지 반란            |

## 행복의 원천 / 감소

### + 원천

| 원천        | 출처                                     |
| --------- | -------------------------------------- |
| 사치품 보유    | [사치품](docs/table/아이템/사치품.md) — 종류 다양성 보너스 |
| 복지·문화 건물  | 우물 · 신전 · 광장 · 술집 ([건물](docs/table/세력/건물.md)) |
| 다양한 식량 공급 | [자원](docs/table/아이템/자원.md)의 `행복` 컬럼    |
| 총독 매력     | [유닛](docs/table/세력/유닛.md)의 `매력`        |

### − 감소

| 감소 요인       | 조건                        |
| ----------- | ------------------------- |
| 인구 과밀       | 인구 > 집·우물 등 수용 한계         |
| 식량 부족       | 인구 대비 식량 생산 부족            |
| 단조로운 식단     | 한 종류 식량만 소비 (자원 사기 규칙과 동일) |
| 전쟁 패배 · 고세율 | 영지 인근 패전, 세율 과다           |

## 계산식 (초안)

```
영지 행복 = 기본(50)
           + Σ(건물 행복)
           + Σ(사치품 행복 · 다양성)
           + Σ(식량 다양성)
           + 총독 매력 / 10
           − 과밀 페널티
           − 식량부족 페널티
           → clamp(0 ~ 100)

생산 배율 = clamp(0.8 ~ 1.2, 1 + (행복 − 50) / 250)
```

- **다양성**: 같은 품목을 쌓기보다 서로 다른 종류를 보유할수록 보너스가 커진다.
- 위 계수(50·250·0.8~1.2 등)는 밸런싱용 초안값이며 조정 대상이다.
