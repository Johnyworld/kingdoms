# 승리 조건

한 판(캠페인/시나리오)의 종료 조건. 승리·패배 조건을 정의한다.

- **세력 소멸**: 어떤 [세력](docs/table/세력/세력.md)이 지휘소([캠프·마을회관·성](docs/table/세력/건물.md))를 하나도 보유하지 않으면 소멸한다. 소속 [영지](docs/table/세력/영지.md)·부대도 함께 붕괴한다.

### 승리 조건

| 조건       | 모드   | 달성 방법                              |
| -------- | ---- | ---------------------------------- |
| 정복 승리    | 기본   | 플레이어를 제외한 모든 세력을 **소멸**시킨다         |
| 거점 점령    | 시나리오 | 지정된 목표 영지(수도·요새)를 점령한다             |
| 기간 내 우위  | 선택   | 정해진 턴이 지난 시점에 **영지 수·점수 1위**를 유지한다 |

### 패배 조건

| 조건     | 발생                                    |
| ------ | ------------------------------------- |
| 세력 소멸  | 플레이어의 모든 지휘소(영지)를 상실                  |
| 영웅 사망  | 주인공(영웅) [유닛](docs/table/세력/유닛.md)이 사망 |

### 점수 *(초안, 기간 모드용)*

```
점수 = 영지 수 × 100 + 인구 + 보유 자원 환산 + 처치한 적 부대 × 50
```

- 세부 가중치·기간·거점 지정 등은 시나리오별로 설정한다. *(초안)*
