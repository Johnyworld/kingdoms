# 이벤트

특정 계기로 발생하는 상황. 탐사·조우·랜덤·스토리로 나뉜다.

### 속성

| 속성명    | 타입                             | 설명              |
| ------ | ------------------------------ | --------------- |
| 이름     | string                         |                 |
| 분류     | 탐사 \| 조우 \| 랜덤 \| 스토리          |                 |
| 트리거    | string                         | 발생 계기           |
| 조건     | string                         | 발생·판정에 필요한 조건   |
| 결과/선택지 | string                         | 결과 또는 플레이어 선택지  |

### 일람

| 분류  | 이름       | 트리거              | 조건          | 결과/선택지                    |
| --- | -------- | ---------------- | ----------- | ------------------------- |
| 탐사  | 폐허 탐사    | 폐허 진입            | -           | 아래 **탐사 결과표**             |
| 탐사  | 고대 유적    | 고대 유적 진입         | -           | 희귀 아이템·상아·다이아몬드 / 수호자 전투  |
| 탐사  | 보물 상자    | 보물 상자 진입         | -           | 금·아이템 (드물게 함정)            |
| 조우  | 방랑 영웅    | 중립 마을·특정 헥스 접촉   | 매력/설득 판정    | 성공 시 영웅 [유닛](docs/table/세력/유닛.md) 영입 |
| 조우  | 상인 카라반   | 카라반 접촉           | -           | 임시 상점 개장 ([사치품](docs/table/아이템/사치품.md) 매매) |
| 랜덤  | 풍작       | 매 턴 확률           | 농장 보유       | 식량 +, [행복](docs/table/시스템/행복.md) +   |
| 랜덤  | 가뭄       | 매 턴 확률           | -           | 식량 −, 행복 −                |
| 랜덤  | 역병       | 매 턴 확률 (과밀 시 상승) | -           | 인구·생명점 −, 치료소 보유 시 완화     |
| 랜덤  | 도적 출몰    | 매 턴 확률           | -           | [도적 캠프](docs/table/시스템/월드맵%20오브젝트.md) 생성 |
| 스토리 | 선전포고     | 세력 관계 악화         | -           | 특정 세력이 플레이어에 적대           |

### 탐사 결과표 *(초안)*

폐허 등 [유적](docs/table/시스템/월드맵%20오브젝트.md) 탐사 시 가중 확률로 결정한다.

| 결과   | 확률  | 내용                       |
| ---- | --- | ------------------------ |
| 보물   | 40% | 자원·금 획득                  |
| 유물   | 25% | 아이템·[장신구](docs/table/아이템/장신구.md) 획득 |
| 빈터   | 20% | 소득 없음                    |
| 함정   | 10% | 탐사 부대 피해 + [상태이상](docs/table/시스템/상태이상.md) |
| 수호자  | 5%  | 강한 중립 부대와 [전투](docs/table/시스템/전투.md), 승리 시 큰 보상 |
