# 인공지능

AI는 두 층위로 나뉜다. **전투 AI**는 [전투](docs/table/시스템/전투.md)씬에서 유닛 개개인의 행동을, **전략 AI**는 월드맵에서 세력·부대의 의사결정을 담당한다.

| 층위    | 주체        | 판단 대상               |
| ----- | --------- | ------------------- |
| 전투 AI | 유닛 개개인    | 이동 · 목표 선정 · 교전 · 시전 |
| 전략 AI | 세력 · 부대   | 확장 · 침공 · 방어 · 건설    |

## 전투 AI (전투씬)

플레이어가 조작하지 않는 [부대](docs/table/세력/부대.md)의 각 유닛은 아래 규칙으로 **개별** 행동한다.

### 행동 순서

- 유닛 행동 순서는 **민첩** 내림차순. 동률 시 근접거리(리치) 높은 쪽 우선. ([전투](docs/table/시스템/전투.md))

### 목표 선정 우선순위

1. **원거리 유닛**: 사거리(무기 `공격거리`) 안의 적을 우선 노린다. 사거리 밖이면 대기 또는 위치 조정.
2. **근접 유닛**: **가장 가까운** 적에게 돌진한다.
3. 동일 조건이면 아래 순서로 가점:
   - 처치 가능(예상 피해 ≥ 대상 생명점)한 적
   - 원거리·마법 딜러 등 위협도 높은 적
   - 이미 아군과 교전 중이라 협공 가능한 적
4. 교전 중 **목표 사망 시 즉시 재탐색**한다.

### 병종별 행동

| 병종     | 행동                                              |
| ------ | ----------------------------------------------- |
| 근접     | 목표에게 돌진 → 접촉 시 교전(3회 공방). 리치 긴 쪽 선제.            |
| 원거리    | **원거리 공격 모드에서만** 발사, 근접 교전 중에는 대기. 반격은 사거리 내에서만. |
| 마법     | 사거리 안에서 가장 효과적인 [마법](docs/table/마법/불마법.md) 스킬 시전. 스태미나가 부족하면 저비용 스킬/평타로 전환. |
| 지휘관    | 되도록 **지휘 범위**를 유지해 아군 보너스를 지키며, 무리한 단독 돌격을 피한다. |

### 도주 · 회피

- 사기가 임계 이하이거나 **공포** [상태이상](docs/table/시스템/상태이상.md)이면 확률적으로 도주한다.
- 지휘관이 사망하면 부대 전체의 사기가 떨어져 도주 확률이 오른다.

## 전략 AI (월드맵)

비플레이어 세력이 턴마다 내리는 의사결정.

### 세력 성향 *(초안)*

세력별로 가중치가 다른 성향을 둔다. — [세력](docs/table/세력/세력.md)

| 성향   | 특징                          |
| ---- | --------------------------- |
| 확장형  | 캠프 건설·영지 확보를 우선             |
| 공격형  | 적 부대·영지 침공을 우선             |
| 방어형  | 자국 영지 방어·성벽/방어탑 건설을 우선     |
| 경제형  | 자원·생산 건물과 시장·상업을 우선         |

### 의사결정 우선순위 *(초안)*

1. **위기 대응**: 자국 영지가 위협받으면 방어 부대를 회군시킨다.
2. **생산·경제**: 여유인구·자원으로 필요한 [건물](docs/table/세력/건물.md)을 건설(선행 조건·자원 충족 순).
3. **군사**: 병력이 충분하면 성향에 따라 침공/확장 부대를 편성.
4. **확장**: 빈 요지에 캠프를 세워 새 영지를 확보.

### 부대 운용

- **방어 부대**: 영지 인근에 배치, 시야·정찰 유지.
- **침공 부대**: 목표 영지·적 부대로 진군. [전투](docs/table/시스템/전투.md) 유리도(병력·지형·상성)를 평가해 교전 여부 결정.
- **정찰**: 이동력·시야가 높은 유닛으로 안개를 밝힌다.
- 이동 시 [타일](docs/table/시스템/타일.md) 스태미나·이동력 보정을 고려한 경로를 선택한다.
