# 인트로

## 타이틀 메뉴

| 항목       | 설명                          |
| -------- | --------------------------- |
| 새 게임     | 배경 시나리오 시작 (아래 새 게임 흐름)      |
| 이어하기     | 저장된 게임 불러오기                  |
| 설정       | 옵션                          |
| 신규 영웅 관리 | 새 게임에 쓸 주인공(영웅) 프리셋 생성·편집    |
| 신규 세력 관리 | 새 게임에 쓸 플레이어 세력 프리셋 생성·편집    |

## 새 게임 흐름

1. **영웅(주인공) 생성** — 주인공은 [유닛](docs/table/세력/유닛.md) 객체로 만들어진다. 이름·성별·종족([인간](docs/table/세력/종족.md))·특성을 설정.
2. **세력 생성** — 플레이어 [세력](docs/table/세력/세력.md)을 새로 만든다(이름·색상). NPC 4대 세력과 별개다.
3. **시작 배치** — [캠프](docs/table/세력/건물.md) 1채로 시작해 첫 [영지](docs/table/세력/영지.md)를 확보한다. 시작 지형은 [타일](docs/table/시스템/타일.md)별 규칙을 따른다.

## 배경 *(초안)*

> **때는 왕국력 200년.** 대륙은 오랜 균형 속에 네 강대 세력이 저마다의 땅을 지배한다.

| 세력    | 색  | 성격 (초안)                     |
| ----- | -- | --------------------------- |
| 푸른 왕국 | 청 | 오래된 기사도의 정통 왕국. 질서와 방어를 중시   |
| 사막 술탄국 | 적 | 교역과 부로 번영한 사막의 상업 국가         |
| 암흑 제국 | 자 | 금단의 마법을 탐구하는 팽창주의 제국         |
| 초원 칸국 | 녹 | 말과 기동전에 능한 유목 기마 세력          |

네 세력이 서로를 견제하는 사이, 이름 없는 한 **신흥 군주(주인공)**가 변경의 작은 캠프에서 일어선다.
캠프를 마을회관으로, 다시 성으로 키우고, 영지를 넓히며, 부대를 조직해 대륙의 판도에 뛰어든다.

- **시작**: 캠프 1채, 소수의 인원, [초기 자원](docs/table/세력/영지.md)(금50·빵30·밀30·목재10·석재10·철괴10).
- **목표**: 영지 확장과 세력 규합을 통해 대륙의 패권을 잡는다. 상세는 [승리 조건](docs/table/시스템/승리조건.md) 참조.
