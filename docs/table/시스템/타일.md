### 속성

| 속성명         | 타입            |
| ----------- | ------------- |
| 이름          | string        |
| 이동타입        | 육지 \| 하늘 \| 배 |
| 스태미나 소모(육지) | number        |
| 이동력 보정      | number        |
| 방어 보정       | number        |
| 시야 보정       | number        |
| 건설 가능       | bool          |

### 일람

| 이름  | 이동타입      | 스태미나(육지) | 이동력 보정 | 방어 보정 | 시야 보정 | 건설  |
| --- | --------- | -------- | ------ | ----- | ----- | --- |
| 평원  | 육지, 하늘    | 2        | 0      | 0     | 0     | O   |
| 도로  | 육지, 하늘    | 1        | +2     | -1    | 0     | X   |
| 수풀  | 육지, 하늘    | 3        | 0      | +1    | -1    | O   |
| 숲   | 육지, 하늘    | 3        | 0      | +2    | -2    | O   |
| 습지  | 육지, 하늘    | 4        | 0      | -1    | 0     | X   |
| 사막  | 육지, 하늘    | 4        | 0      | 0     | +1    | O   |
| 설원  | 육지, 하늘    | 4        | 0      | 0     | +1    | O   |
| 언덕  | 육지, 하늘    | 3        | -1     | +2    | +2    | O   |
| 해안  | 육지, 하늘, 배 | 3        | 0      | -1    | 0     | O   |
| 다리  | 육지, 하늘    | 2        | 0      | -1    | 0     | X   |
| 강   | 하늘, 배     | -        | -      | 0     | 0     | X   |
| 물   | 하늘, 배     | -        | -      | 0     | 0     | X   |
| 산   | 하늘        | -        | -      | +3    | +3    | X   |

### 규칙

- 스태미나 소모 수치는 하늘·배에는 적용되지 않음.
- 하늘 유닛은 모든 지형에서 2.
- 배는 물·강·해안에서만 이동하며 모든 지형에서 3.
- **이동력 보정**은 해당 타일에 올라선 유닛의 이동력에 가감된다. 도로 +2, 언덕 -1.
- **방어/시야 보정**은 전투·시야 계산에 가감된다. (숲은 방어 +2·시야 -2, 산·언덕은 시야 확보)
- **건설 X** 타일(도로·습지·다리·강·물·산)에는 건물을 지을 수 없다.
- 언덕은 광산 계열(철광·금광·은광) 배치의 근거 지형이며, 산은 지상 통행이 불가하다.
- 강에는 다리를 놓아 육지 유닛이 건널 수 있다.
