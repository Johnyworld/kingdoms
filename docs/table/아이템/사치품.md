### 속성

| 속성명  | 타입                            | 설명                     |
| ---- | ----------------------------- | ---------------------- |
| 이름   | string                        |                        |
| 분류   | 기호식품 \| 예술품 \| 보석 \| 진귀품      |                        |
| 행복   | number                        | 영지 보유 시 인구 행복 증가       |
| 효과   | string                        | 행복 외 추가 효과             |
| 획득   | 제작 \| 상인 \| 채집                | 주 획득 경로                |
| 무게   | number                        |                        |
| 가치   | number                        |                        |

- 사치품은 소비되지 않고, 영지/부대가 **보유하는 동안** 행복을 제공한다.
- 같은 품목을 쌓기보다 **여러 종류를 보유할수록 행복 보너스가 커진다(다양성)**. — [자원](docs/table/아이템/자원.md)의 "다양한 음식 → 사기" 규칙과 같은 사상.
- 상인/시장에서 주로 거래되며 가치가 높다.

### 규칙

- **보석**은 [보석 세공소](docs/table/세력/건물.md)의 원료가 되어 [장신구](docs/table/아이템/장신구.md)로 가공되며, 일부는 마법 무기(오브·완드) 촉매로도 쓰인다.
- **비단**은 상위 천옷 방어구, **상아·다이아몬드**는 고급 장신구 재료로 연결된다.
- 채집형 사치품(보석·차·소금·모피 등)은 특정 지형·[월드맵 오브젝트](docs/table/시스템/월드맵%20오브젝트.md)에서 획득한다.
