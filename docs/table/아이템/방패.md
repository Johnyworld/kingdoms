### 속성

| 속성명   | 타입                                         | 설명                              |
| ----- | ------------------------------------------ | ------------------------------- |
| 이름    | string                                     |                                 |
| 방어력   | number                                     | 착용 시 방어력에 합산                    |
| 막기 확률 | number                                     | 공격을 완전히 막을 확률(%)                |
| 무게    | number                                     |                                 |
| 필요직업  | [직업](docs/table/세력/직업.md)                  | 제작 담당 직업                         |
| 생산비용  | NumberWithList<[자원](docs/table/아이템/자원.md)> |                                 |
| 가치    | number                                     |                                 |

- 방패는 한 손을 점유한다. **양손무기**와 함께 착용할 수 없다.

### 일람

| 이름      | 방어력 | 막기 확률 | 무게  | 필요직업 | 생산비용     | 가치  |
| ------- | --- | ----- | --- | ---- | -------- | --- |
| 나무 방패   | 3   | 20    | 2   | 목수   | 목재3      | 8   |
| 버클러     | 2   | 15    | 1   | 대장장이 | 철괴1      | 12  |
| 라운드 실드  | 5   | 25    | 3   | 대장장이 | 목재2, 철괴1 | 20  |
| 카이트 실드  | 8   | 30    | 5   | 대장장이 | 목재2, 철괴2 | 32  |
| 타워 실드   | 12  | 40    | 8   | 대장장이 | 목재2, 철괴4 | 48  |

### 규칙

- **버클러**: 가장 가볍고 기동성이 좋으나 막기 확률이 낮다. 결투·경보병용.
- **라운드 실드**: 표준형. 무게 대비 균형이 좋다.
- **카이트 실드**: 몸을 크게 가려 원거리 방어에 유리하다.
- **타워 실드**: 최고의 방어력·막기 확률이나 무거워 이동력·회피에 불리하다. 방패병용.
- 막기 성공 시 해당 공격 피해를 무효화한다. 실패 시 방어력만 적용된다.
