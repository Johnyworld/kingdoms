### 속성

| 속성명   | 타입                                         | 설명           |
| ----- | ------------------------------------------ | ------------ |
| 이름    | string                                     |              |
| 분류    | 회복 \| 버프 \| 공격 \| 해제                       |              |
| 효과    | string                                     |              |
| 대상    | 자신 \| 아군 \| 적 \| 광역                        |              |
| 지속    | number                                     | 지속 턴 (0 = 즉시) |
| 필요직업  | [직업](docs/table/세력/직업.md)                  | 제작 담당 직업      |
| 생산비용  | NumberWithList<[자원](docs/table/아이템/자원.md)> |              |
| 무게    | number                                     |              |
| 가치    | number                                     |              |

- 소모품은 1회 사용 후 소멸한다.

### 규칙

- **지속 0** = 즉시 발동. 그 외에는 지속 턴 동안 효과가 유지된다.
- 화상·중독 등 부여/해제 효과는 [상태이상](docs/table/시스템/상태이상.md) 정의를 따른다.
- 제작처: 회복·버프·공격 물약은 **연금술 공방(연금술사)**, 붕대·해독제는 **치료소(의사)**.
- 음식(밀·빵·고기 등)은 소모품이 아니라 [자원](docs/table/아이템/자원.md)으로 관리하며, 사기에 영향을 준다.
