### 속성

| 속성명   | 타입                                         | 설명                |
| ----- | ------------------------------------------ | ----------------- |
| 이름    | string                                     |                   |
| 분류    | 반지 \| 목걸이 \| 귀걸이 \| 부적                     | 부위별 슬롯            |
| 효과    | string                                     | 착용 시 스탯·마법·저항 보정  |
| 생산비용  | NumberWithList<[자원](docs/table/아이템/자원.md) · [사치품](docs/table/아이템/사치품.md)> |                   |
| 무게    | number                                     |                   |
| 가치    | number                                     |                   |

- [보석 세공소](docs/table/세력/건물.md)에서 **보석세공사**가 제작한다.
- 재료의 [보석](docs/table/아이템/사치품.md)(루비·사파이어·에메랄드·다이아몬드)·상아는 사치품에서 조달한다.
- 분류별로 슬롯이 하나씩. 유닛은 반지·목걸이·귀걸이·부적을 각 1개 착용한다.

### 규칙

- 마법 보정 장신구의 수치는 해당 속성 [마법](docs/table/마법/불마법.md) 기능치에 합산되어 위력을 높인다.
- 저항 계열(정화·용기의 부적)은 [상태이상](docs/table/시스템/상태이상.md) 부여 확률·지속을 줄인다.
- 착용 효과는 [전투](docs/table/시스템/전투.md)의 계산 스탯 산출 시 함께 반영된다.
