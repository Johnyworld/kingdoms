# 얼음 마법

제어 특화 계열. **둔화**와 **빙결**로 적의 행동을 묶는다. 피해 계산은 [전투](docs/table/시스템/전투.md)의 마법 피해식을 따른다.

- 시전 위력 = 촉매([무기](docs/table/아이템/무기.md)) 공격력 + 시전자의 `얼음 마법` 기능치
- 부여 상태이상: [둔화 · 빙결](docs/table/시스템/상태이상.md)
