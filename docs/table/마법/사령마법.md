# 사령 마법

저주 특화 계열. **공포·저주·침묵·중독**으로 적을 약화시키고 언데드를 부린다. 피해 계산은 [전투](docs/table/시스템/전투.md)의 마법 피해식을 따른다.

- 시전 위력 = 촉매([무기](docs/table/아이템/무기.md)) 공격력 + 시전자의 `사령 마법` 기능치
- 부여 상태이상: [공포 · 저주 · 침묵 · 중독](docs/table/시스템/상태이상.md)
