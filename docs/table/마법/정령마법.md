# 정령 마법

버프·회복 특화 계열. **재생·보호막**으로 아군을 강화한다. 피해 계산은 [전투](docs/table/시스템/전투.md)의 마법 피해식을 따른다.

- 시전 위력 = 촉매([무기](docs/table/아이템/무기.md)) 공격력 + 시전자의 `정령 마법` 기능치
- 부여 상태이상: [재생 · 보호막](docs/table/시스템/상태이상.md)

### 스킬 일람

| ------ | --- | ------ | ---- | --- | ----- | -------- | ---- |
