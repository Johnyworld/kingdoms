# 바람 마법

기동 특화 계열. **실명**과 아군 기동 강화, 밀어내기로 전장을 흔든다. 피해 계산은 [전투](docs/table/시스템/전투.md)의 마법 피해식을 따른다.

- 시전 위력 = 촉매([무기](docs/table/아이템/무기.md)) 공격력 + 시전자의 `바람 마법` 기능치
- 부여 상태이상: [실명 · 신속](docs/table/시스템/상태이상.md)
