# 무기 스킬

물리 무기로 쓰는 액티브 기술. [전투](docs/table/시스템/전투.md) 교전 중 스태미나를 소모해 발동한다.
[마법](docs/table/마법/불마법.md)이 `마법 기능치`로 해금되는 것과 달리, 무기 스킬은 **무기 분류 + 필요 스탯(힘·민첩)**으로 해금된다.

- 피해·명중·치명타·상태이상 판정은 모두 [전투](docs/table/시스템/전투.md) 규칙을 따른다.
- 데미지 타입 → 치명타 상태이상 연동: 참격→출혈, 타격→기절. ([상태이상](docs/table/시스템/상태이상.md))

### 스킬 일람

| 스킬     | 무기 분류 | 필요 스탯 | 스태미나 | 대상     | 효과                | 상태이상 |
| ------ | ----- | ----- | ---- | ------ | ----------------- | ---- |
| 강타     | 검     | 힘 20  | 4    | 단일     | 피해 ×1.5           | -    |
| 연속 베기  | 검     | 힘 40  | 6    | 단일     | 추가 2회 타격          | -    |
| 급소 찌르기 | 검     | 힘 55  | 7    | 단일     | 치명타 확정            | 출혈   |
| 분쇄     | 도끼    | 힘 30  | 5    | 단일     | 대상 방어력 −30% (2턴)  | -    |
| 회전 베기  | 도끼    | 힘 45  | 7    | 광역(인접) | 주변 적 동시 타격        | -    |
| 관통     | 창     | 힘 25  | 5    | 단일     | 방어력 무시            | -    |
| 견제     | 창     | 민첩 25 | 4    | 단일     | 리치 선제 + 밀어내기      | -    |
| 후려치기   | 둔기    | 힘 30  | 5    | 단일     | 피해 + 기절 확률↑       | 기절   |
| 방패 부수기 | 둔기    | 힘 35  | 5    | 단일     | 대상 방패 막기 무효 (2턴)  | -    |
| 조준 사격  | 활     | 민첩 30 | 5    | 단일     | 명중 +, 치명타 +30%    | -    |
| 다중 사격  | 활     | 민첩 45 | 7    | 다수     | 사거리 내 여러 목표 타격    | -    |
| 약점 사격  | 활     | 민첩 55 | 8    | 단일     | 방어력 절반 무시         | 출혈   |
| 돌진     | 근접 전체 | 힘 20  | 4    | 단일     | 이동 후 즉시 공격        | -    |
| 방어 태세  | 전체    | -     | 3    | 자신     | 이번 턴 받는 피해 −50%   | -    |
