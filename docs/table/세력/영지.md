### 속성

| 속성명  | 타입                                         |
| ---- | ------------------------------------------ |
| 이름   | string                                     |
| 인구   | number                                     |
| 여유인구 | number                                     |
| 자원   | NumberWithList<[자원](docs/table/아이템/자원.md)> |
| 총독   | [유닛](docs/table/세력/유닛.md)                  |
| 세력   | [세력](docs/table/세력/세력.md)                  |


### 일람

```
const 초기자원 = 금50, 빵30, 밀30, 목재10, 석재10, 철괴10
```

| 이름    | 인구  | 여유인구 | 자원     | 총독       | 세력     |
| ----- | --- | ---- | ------ | -------- | ------ |
| 창천성   | 10  | 10   | {초기자원} | 아젤 하르윈   | 푸른 왕국  |
| 알사바흐  | 10  | 10   | {초기자원} | 카심 이븐 라시드 | 사막 술탄국 |
| 흑요요새  | 10  | 10   | {초기자원} | 발타자르     | 암흑 제국  |
| 텡그리 언덕 | 10  | 10   | {초기자원} | 바트르 칸    | 초원 칸국  |
