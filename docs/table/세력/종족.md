### 속성

| 속성명   | 타입     | 설명                   |
| ----- | ------ | -------------------- |
| 이름    | string |                      |
| 이동력   | number | 기본 이동력 (도보 기준)       |
| 시야    | number | 기본 시야                |
| 스탯 보정 | string | 특성치 가감               |
| 특성    | string | 종족 고유 성향·능력          |

### 일람

| 이름  | 이동력 | 시야  | 스탯 보정 | 특성                            |
| --- | --- | --- | ----- | ----------------------------- |
| 인간  | 4   | 7   | -     | 균형형. 모든 직업·마법 습득 가능, 성장이 빠르다  |
