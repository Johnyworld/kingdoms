=== CONTENTS ===
- 25 unique UI assets
- Packaged as tilesheet PNG (384x352 px)

=== USAGE TIPS ===
- Asset art uses vector graphics and seamlessly scales above a 32x32 tilesize.
- Slice tiles 16x16 if you need a smaller tilesize instead of full scaling.
- Scaling assets BELOW the original size (based on 32x32 tiles) is not recommended.
- Palette hex codes for custom color matching: 
    Honey Gold: #dcc47c
    Tan: #9f8873
    Camel: #C19149
    Charcoal Brown: #2E322A
    Powder Blue: #406c91
    Dark Blue: #15202f
